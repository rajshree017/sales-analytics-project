# Sales Data Analysis using Python

## Project Overview
This project focuses on analyzing sales data using Python to uncover meaningful insights across customer demographics, regions, and product categories. The analysis helps understand purchasing behavior and sales trends for data-driven decision making.

## Dataset
The dataset contains customer-level sales information including gender, age group, state, occupation, product category, number of orders, and purchase amount.

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Key Analysis Performed
- Data cleaning and preprocessing (handling missing and unwanted columns)
- Exploratory Data Analysis (EDA)
- Sales analysis by gender, age group, marital status, and state
- Product category and occupation-wise sales analysis
- Data visualization using bar charts and pie charts

## Key Insights
- Identified sales distribution across different demographic groups
- Analyzed top-performing product categories and regions
- Compared purchase behavior based on gender and marital status

## Conclusion
This project demonstrates practical data analytics skills including data cleaning, EDA, and visualization to extract actionable business insights from raw sales data.
